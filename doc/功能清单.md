# 宠物管理系统功能需求

## 功能概述

本项目是一个针对宠物主人的信息管理Web系统，帮助用户科学管理宠物的健康和日常护理。具体实现以下功能：

**用户系统**：用户可以注册账号、登录系统、上传头像、管理个人信息，支持一个账号管理多只宠物。

**宠物档案**：录入宠物基本信息（品种、年龄、性别、照片等），支持增删改查操作，用户可以随时查看和更新宠物资料。

**健康记录**：记录疫苗接种时间、体检结果、疾病历史等健康数据，自动计算下次疫苗接种时间，提供完整的健康管理档案。

**日常记录**：简单记录每日喂食时间、食物类型、运动时长、运动类型、洗澡日期等日常护理信息，方便回顾和统计。

**医院管理**：手动添加常去的宠物医院信息，记录每次就诊情况、诊断结果、费用支出，建立就诊历史档案。

**提醒功能**：用户可以设置疫苗提醒、体检提醒、洗澡提醒等，系统通过站内消息进行通知。

**数据展示**：使用图表展示宠物体重变化趋势、疫苗接种统计、护理费用统计等数据，让管理更直观。

**AI助手**：集成免费的智谱GLM API，用户可以咨询简单的宠物健康问题，获得基础的护理建议和常见问题解答。

系统使用Vue.js前端框架和Spring Boot后端框架开发，MySQL数据库存储，采用前后端分离的现代Web开发模式。
